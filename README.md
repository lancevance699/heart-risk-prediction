# Heart Attack Risk Prediction

## Описание проекта
В этом проекте разработана модель машинного обучения для прогнозирования риска сердечного приступа на основе медицинских и поведенческих факторов.

Проект выполнен в рамках учебной практики и ориентирован на полный цикл работы с задачей бинарной классификации: от анализа данных до интерпретации результатов модели.

## Цель проекта
Построить модель, способную определять риск сердечного приступа по набору признаков, связанных с состоянием здоровья, образом жизни и физиологическими показателями пациента.

## Использованные данные
В работе использовались два датасета:
- `heart_train.csv` — обучающая выборка с целевым признаком
- `heart_test.csv` — тестовая выборка без целевого признака

Целевой признак:
- `Heart Attack Risk (Binary)` — бинарный индикатор риска сердечного приступа

## Этапы работы
В рамках проекта были выполнены следующие шаги:
1. Загрузка и первичный анализ данных
2. Проверка структуры данных, пропусков и дубликатов
3. Предобработка данных:
   - удаление служебных признаков
   - заполнение пропусков медианой
   - кодирование признака `Gender`
4. Исследовательский анализ данных с визуализацией
5. Обучение модели классификации `CatBoostClassifier`
6. Оценка качества модели по метрикам:
   - ROC-AUC
   - Accuracy
   - F1-score
7. Подбор порога классификации
8. Анализ важности признаков
9. Сохранение обученной модели

## Результаты
Базовая модель показала следующие результаты на валидационной выборке:
- ROC-AUC: 0.57
- Accuracy: 0.66
- F1-score: 0.16

После подбора порога классификации удалось улучшить качество выявления положительного класса:
- F1-score: 0.50
- Accuracy: 0.48

Снижение accuracy в данном случае является допустимым, так как в медицинских задачах важнее повысить чувствительность модели к потенциально рискованным случаям.

## Наиболее важные признаки
Наибольшее влияние на предсказание модели оказали:
- Diet
- Cholesterol
- Triglycerides
- Sedentary Hours Per Day
- Exercise Hours Per Week
- BMI
- Age
- Income
- Systolic blood pressure
- Heart rate
